# Rook CLI

Rook provides a terminal-first interface for AI-assisted work.

## Start the TUI

```bash
rook
```

## Start or resume a session

```bash
rook session
rook session --resume
```

## Run a one-off instruction

```bash
rook run --text "Summarize this project"
```

## Configure Rook

```bash
rook configure
```

## Check setup

```bash
rook doctor
```
