# Quickstart

Rook is a local AI agent for terminal and desktop workflows.

## Build from source

```bash
git clone https://github.com/ShaileshRawat1403/rook.git
cd rook
source bin/activate-hermit
cargo build -p rook-cli
```

## Run Rook

```bash
./target/debug/rook
```

Running `rook` with no arguments opens the Rook terminal interface.

## CLI help

```bash
./target/debug/rook --help
```
