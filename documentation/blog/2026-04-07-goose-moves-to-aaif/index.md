Block has donated rook to the [Agentic AI Foundation (AAIF)](https://aaif.io/) at the Linux Foundation, alongside Anthropic's Model Context Protocol (MCP) and OpenAI's AGENTS.md. You can read the [full announcement here](https://aaif.io/press/linux-foundation-announces-the-formation-of-the-agentic-ai-foundation-aaif-anchored-by-new-project-contributions-including-model-context-protocol-mcp-rook-and-agents-md/).

While nothing changes with rook's dedication to its mission and community, here's what's changed;

<!-- truncate -->

## New Documentation site
The documentation has moved to a new site: [https://rook-docs.ai/](https://rook-docs.ai/) with a nice .ai TLD! The old docs links will redirect, but update your bookmarks to the new site.

## Updated GitHub repository
The GitHub repository and related projects have moved from `block/rook` to a new org:

> **[https://github.com/aaif-rook/rook](https://github.com/aaif-rook/rook)**

This includes related repositories in the [aaif-rook](https://github.com/aaif-rook) organization.

Everything else — the project, the community, the mission — remains the same. rook is still open source, still actively developed, and still the same agent you know.

## Update your remotes

If you have a local clone, update your git remote:

```bash
git remote set-url origin git@github.com:aaif-rook/rook.git
```

## Migration in progress

We're still working through some migration issues (broken links, redirects, CI, etc). If you hit anything that seems off, please reach out on [Discord](https://discord.gg/rook-oss) and let us know.
## Learn more

Visit [aaif.io](https://aaif.io/) to learn more about the foundation and its mission.
